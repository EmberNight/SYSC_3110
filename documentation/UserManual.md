# User Manual

SYSC 3110 - Software Development Project - Term Project

Author: Tanner Trautrim

## Usage

1. Open your desired terminal
2. Navigate to the jar location
3. Run the jar file (replace the contents in the quotations). ```java -jar "name of jar file"```
4. Enter the number of players
5. Start playing
6. View Game Status on the right

## Modes

### Allocate Armies

1. Select a territory on the left
2. Click add units
3. Enter the number of units you want to place
4. repeat until done

### Attack

1. Select a territory on the left
2. Select a territory on the right
3. Click the attack button
4. View the results
5. repeat until done

### Attack

1. Select a territory on the left
2. Select a territory on the right
3. Click the move units button
4. repeat until done

### Next Phase

1. Click "Next Phase" to switch phases
2. Phase 1 is allocate armies
3. Phase 2 is attack
4. Phase 3 is move armies
5. Phase 4 is next players turn
